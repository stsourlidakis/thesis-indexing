Πληροφορίες

Σε αυτό το website θα βρείτε πληροφορίες για παλιότερες πτυχιακές φοιτητών του ΤΕΙ Στερεάς Ελλάδας. Τα δεδομένα προέρχονται από την ιστοσελίδα της βιβλιοθήκης που δυστυχώς έχει να ενημερωθεί καιρό.

Αναπτύχθηκε και συντηρείται από τον Σταύρο Τσουρλιδάκη και ο πηγαίος κώδικας βρίσκεται στο GitHub σαν λογισμικό ανοιχτού κώδικα υπό την άδεια MIT.

Για οποιαδήποτε πρόταση, σχόλιο ή πρόβλημα στείλτε email στο s.tsourlidakis@hotmail.com ή ανοίξτε ένα νέο GitHub issue.
